## 분석

### 수익성 분포

- **7개 코인 모두 수익 발생** (best config 기준)
- 평균 수익률: 154.8% (HC 미적용)
- 평균 MDD: 47.7%
- **수익 발생 설정 비율**: 52-87% (코인별 상이)

### 코인별 특성

#### Tier 1: 고수익 (>200%)
- **SOLUSDT**: 최고 수익 257%, 높은 변동성(MDD 60%)으로 공격적 트레이딩에 적합
- **ETHUSDT**: 255% 수익, 상대적으로 낮은 MDD(46%), HC 적용시 최고 Profit/MDD(3.49)

#### Tier 2: 중수익 (100-200%)
- **XRPUSDT**: 200% 수익, 높은 MDD(60%), 느린 MA(50) 선호
- **ORDIUSDT**: 146% 수익, 짧은 데이터(2.5년)에도 양호한 성과

#### Tier 3: 저수익 (<100%)
- **GMTUSDT**: 100% 수익이지만 낮은 MDD(35%), Profit/MDD 2.86으로 안정적
- **CETUSUSDT**: 70% 수익, 짧은 데이터(1.5년), 안정적 운용 가능
- **RENDERUSDT**: 55% 수익, 가장 짧은 데이터(1.7년), 추가 검증 필요

### 공통 패턴

1. **MA Period**: 25-50이 최적 (빠른 반응 선호)
2. **Take Profit**: 7-10%가 대부분 (높은 TP 선호)
3. **Stop Loss**: 2-5% (타이트~중간)
4. **Half Close**: 대부분 Profit/MDD 개선 효과

## 결론

1. **전략 유효성 확인**: 7개 코인 모두에서 양의 수익 발생
2. **분산 투자 권장**: 단일 코인 집중보다 포트폴리오 구성
3. **Half Close 필수**: 대부분 코인에서 Profit/MDD 개선
4. **BTCUSDT 핵심 유지**: 리스크 조정 기준 여전히 최고 성과
5. **추가 검증 필요**: RENDERUSDT, CETUSUSDT는 데이터 기간이 짧아 추가 모니터링 필요
