## 개요

729개의 파라미터 조합을 테스트하여 5분봉 최적 설정을 탐색했습니다.

### 테스트 범위

| 파라미터 | 테스트 값 |
|:---------|:---------|
| MA Period | 24, 48, 100 |
| Consolidation Bars | 6, 12, 24 |
| Consolidation Range | 1%, 2%, 3% |
| Drop Threshold | 1.5%, 2%, 3% |
| Take Profit | 2%, 3%, 5% |
| Stop Loss | 1%, 1.5%, 2% |

{: .note }
1시간봉 대비 짧은 타임프레임을 고려하여 TP/SL 범위를 축소하고, 횡보 판단 봉수를 확대했습니다.

## 주요 발견

### 1. 짧은 타임프레임의 특성

```
5분봉 특성:
- 노이즈가 많아 긴 MA(100) 효과 감소
- TP/SL 범위를 좁게 설정해야 함
- 횡보 판단에 더 많은 봉수 필요
```

### 2. MA Period 영향

| MA | 특성 |
|:--:|:-----|
| 24 | 빈번한 진입, 낮은 승률, 높은 MDD |
| **48** | **균형잡힌 진입 빈도와 수익** |
| 100 | 적은 진입, 높은 승률, 낮은 수익 |

### 3. 1시간봉 대비 차이

| 항목 | 1시간봉 | 5분봉 |
|:-----|:------:|:----:|
| 최적 MA | 100 | 48 |
| 최적 TP/SL | 10%/5% | 2%/1.5% |
| 수익/MDD | 4.33 | 2.04 |
| 전략 효율 | **높음** | 보통 |

## 결론

{: .warning }
> **5분봉은 1시간봉 대비 효율이 낮습니다.**
>
> - 수익/MDD 비율: 2.04 vs 4.33 (1시간봉의 47%)
> - 노이즈로 인한 낮은 승률
> - ML 예측력 저하 (AUC 0.55 vs 0.62)

**권장**: 
- 메인 전략은 **1시간봉** 사용
- 5분봉은 **진입 타이밍 미세조정** 또는 **스캘핑** 용도로만 활용
