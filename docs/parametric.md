## 주요 발견

### 1. MA 기간의 영향

```
MA25:  빈번한 진입, 낮은 승률, 높은 변동성
MA50:  균형잡힌 진입 빈도와 수익
MA100: 적은 진입, 높은 승률, 안정적 수익
```

{: .important }
**MA100**을 권장합니다. 거래 빈도는 줄지만 품질이 높습니다.

### 2. TP/SL 비율

| TP/SL 비율 | 특성 |
|:---------:|:-----|
| 3.3 (10/3) | 높은 수익, 낮은 승률 |
| 2.0 (10/5) | 균형잡힌 성과 |
| 1.0 (5/5) | 안정적, 낮은 수익 |

### 3. 반익절 효과

| 항목 | 반익절 없음 | 반익절 있음 | 변화 |
|:-----|----------:|----------:|-----:|
| 최고 수익 | 188.6% | 112.5% | -40% |
| 최저 MDD | 36% | 16.5% | -54% |
| 최고 승률 | 59.4% | 88.4% | +49% |
| 수익/MDD | 3.04 | 4.33 | +42% |

## 권장 설정

### 공격적 (수익 극대화)

```python
config = LongStrategyConfig(
    breakout_ma=50,
    take_profit_pct=10.0,
    stop_loss_pct=3.0,
    half_close_enabled=False,
)
# 예상: 188% 수익, 62% MDD
```

### 균형 (권장)

```python
config = LongStrategyConfig(
    breakout_ma=100,
    take_profit_pct=10.0,
    stop_loss_pct=5.0,
    half_close_enabled=True,
    half_close_pct=3.0,
)
# 예상: 112% 수익, 26% MDD
```

### 보수적 (안정성 우선)

```python
config = LongStrategyConfig(
    breakout_ma=100,
    take_profit_pct=5.0,
    stop_loss_pct=5.0,
    half_close_enabled=True,
    half_close_pct=1.0,
)
# 예상: 75% 수익, 16.5% MDD
```
