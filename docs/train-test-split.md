## 분석

### 핵심 인사이트

1. **Out-of-Sample 검증 통과**: 두 코인 모두 2026년에서 양의 수익 달성
   - BTCUSDT: +8.0% (HC 적용)
   - ETHUSDT: +12.6% (HC 적용)

2. **ETH > BTC**: 테스트 기간 동안 ETH가 더 높은 수익률과 Profit/MDD 기록
   - ETH Profit/MDD: 1.33 vs BTC: 0.52

3. **Half Close 효과**: 양 코인 모두 HC 적용 시 MDD 감소, 승률 상승
   - BTC: MDD 21% → 15.5% (-26%)
   - ETH: MDD 15% → 9.5% (-37%)

4. **Overfitting 경고**: Train 수익률 대비 Test 수익률 큰 폭 하락
   - 예상 원인: 5년 학습 기간 vs 4개월 테스트 기간 불균형
   - 추가 검증 필요: Walk-forward 또는 K-fold 시계열 CV
