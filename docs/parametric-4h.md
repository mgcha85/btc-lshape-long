## 개요

729개의 파라미터 조합을 테스트하여 4시간봉 최적 설정을 탐색했습니다.

### 테스트 범위

| 파라미터 | 테스트 값 |
|:---------|:---------|
| MA Period | 25, 50, 100 |
| Consolidation Bars | 3, 5, 7 |
| Consolidation Range | 3%, 5%, 7% |
| Drop Threshold | 5%, 7%, 10% |
| Take Profit | 7%, 10%, 15% |
| Stop Loss | 3%, 5%, 7% |

{: .note }
4시간봉은 1시간봉보다 긴 타임프레임으로, TP/SL 및 Drop Threshold를 확대했습니다.

## 주요 발견

### 1. 타임프레임별 최적 설정 비교

| 항목 | 5분봉 | 15분봉 | 1시간봉 | **4시간봉** |
|:-----|:-----:|:------:|:------:|:----------:|
| 최적 MA | 48 | 25 | 100 | **25** |
| 최적 TP | 2% | 7% | 10% | **10%** |
| 최적 SL | 1.5% | 1.5% | 5% | **7%** |
| Drop Threshold | 2% | 2% | 3% | **5%** |

### 2. 4시간봉의 장단점

**장점:**
- 높은 절대 수익률 (반익절 없이 167%)
- 거래 빈도가 적어 관리 용이
- 노이즈가 적고 추세 파악 용이

**단점:**
- **매우 높은 MDD** (반익절 없이 82%)
- 적은 거래 수로 통계적 신뢰도 낮음
- ML 예측력 부족 (샘플 수 103개)

### 3. 리스크 관리 필수

```
4시간봉은 반익절 없이 운용하면 MDD가 82%에 달합니다.
반드시 반익절 또는 포지션 사이즈 조절이 필요합니다.
```

## 결론

{: .warning }
> **4시간봉은 반익절 없이는 권장하지 않습니다.**
>
> - MDD 82%는 실전에서 감당하기 어려움
> - 반익절 2% 적용으로 MDD를 48%까지 낮출 수 있음
> - 거래 수가 적어 ML 필터 학습이 어려움
> - 1시간봉 대비 수익/MDD 비율이 낮음

**권장**:
- 단독 사용보다 **1시간봉과 조합**하여 확인용으로 활용
- 또는 **반익절 2%** 필수 적용
