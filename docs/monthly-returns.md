## Key Insights

### 1. 연간 패턴
- **2020년**: 두 자산 모두 최고 성과 (COVID 후 상승장)
- **2022년**: 하락장에서 BTC는 큰 손실, ETH는 보합
- **2024년**: 두 자산 모두 회복세

### 2. 전략 특성
- **승률 40% 수준**이지만 **손익비 3:1**로 수익 창출
- **월별 편차 큼** - 좋은 달 +20%, 나쁜 달 -15%
- **Half Close** 적용으로 MDD 관리

### 3. 최적화 제안
- BTC: 하락장(2022) 필터 추가 고려
- ETH: 현재 설정이 더 안정적
- **추천**: ETH 비중 확대 또는 ETH 단독 운용
